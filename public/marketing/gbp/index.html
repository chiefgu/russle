How to use

  1. Open one of the asset pages above.
  2. Use a screenshot tool (CleanShot X, Shottr, or macOS Cmd+Shift+4) to capture just the framed asset rectangle — not the whole page.
  3. Save the resulting PNG.
  4. Upload to GBP via business.google.com → russle → Photos / Logo / Cover photo.
  5. For Posts: business.google.com → russle → Add update → upload the captured image, paste the post copy (see the post copy file for variants), publish.
