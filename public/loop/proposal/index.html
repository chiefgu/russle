04. what it does

Features in detail.

The website is the visible part. Underneath will include.

Referral waitlist
Each signup generates a unique tokenised link. Friends who join via that link increment the original signup's referral count, which moves them up the queue. Position changes trigger an email notification ("you moved from #847 to #623").
Event RSVP capture
Separate forms per event, all writing to the same database with an event_id discriminator. RSVP confirmation emails sent automatically.
Email automation
All transactional emails (welcome, referral notifications, event RSVP confirmations) sent via Resend.
Cookie consent banner
GDPR + UK PECR compliant. Required given Google Analytics is in use.
Admin view
Magic-link authenticated dashboard at a private route. View all signups, sort by position, see referral counts, export to CSV with a documented schema.
SEO + sharing
Open Graph cards for every page (image, title, description), sitemap.xml, robots.txt, custom favicon and app icons.
Analytics
Google Analytics 4 (per the cookie banner) plus Vercel Analytics for performance and Core Web Vitals.
